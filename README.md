# 🌊 InfoKobuleti

Real estate marketplace for Kobuleti, Georgia — [infokobuleti.com](https://infokobuleti.com)

## Stack
- **Backend:** PHP 8.2 (pure MVC, no framework)
- **Database:** MySQL 8.0 (PDO prepared statements)
- **Frontend:** Vanilla CSS + Alpine.js + Phosphor Icons
- **Hosting:** Hostinger (auto-deploy via GitHub Actions)

## Quick Start

```bash
git clone https://github.com/Murvanidz3/infokob.git
cd infokob
# Import database → install/database.sql
# Edit config/config.php with your DB credentials
# Point Apache to this directory
```

## Deployment

Push to `main` → auto-deploys to Hostinger via FTP.

See **[DEPLOYMENT.md](DEPLOYMENT.md)** for setup instructions.

## Features

- 🏠 Property listings with AJAX filtering
- 📱 Mobile-first responsive design
- 🌐 3 languages (Georgian, Russian, English)
- 👤 User registration & listing management
- 🛡️ Admin approval workflow
- ⭐ Featured/premium listings
- 🗺️ Google Maps integration
- 📸 Image upload with auto-resize (3 sizes)
- 💬 WhatsApp, Telegram, Phone contact
- 🔍 SEO optimized with JSON-LD
- 🌊 Kobuleti city guide (CMS)
